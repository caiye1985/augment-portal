## 5. 技术栈映射

### 5.1 前端技术栈
- **框架**：Vue 3.3 + Composition API
- **UI组件**：Element Plus 2.3
- **状态管理**：Pinia 2.1
- **路由管理**：Vue Router 4.x
- **构建工具**：Vite 4.4
- **图表库**：ECharts 5.4
- **HTTP客户端**：Axios

### 5.2 后端技术栈
- **框架**：Spring Boot 3.2
- **语言**：Java 17
- **数据库**：PostgreSQL 15
- **缓存**：Redis 7.0
- **搜索引擎**：Elasticsearch 8.0
- **消息队列**：RabbitMQ 3.12
- **文件存储**：MinIO
- **监控**：Prometheus + Grafana
